## 站立提醒 — 更新日志（2026-03-20 ～ v1.5.0）

> 说明：以下提交在统计区间内均落在 **2026-03-22**（**3 月 20–21 日** 仓库内无新提交）。

### 架构与发布

- **迁移到 Tauri**：替换原 Electron 方案，并清理遗留文件。
- **Windows 构建**：修复构建脚本；安装包支持 **中文 MSI**（及既有 NSIS 流程）。
- **版本**：应用版本提升至 **1.5.0**；增加 `scripts/upload-release-assets.ps1`，便于向 GitHub Release 上传安装包。
- **macOS（Apple Silicon）**：通过 `.github/workflows/build-macos-arm64.yml` 在 GitHub Actions 上构建 **ARM64 / aarch64** DMG（本地 Windows 无法交叉编译 macOS 安装包）。

### 窗口与界面

- **自定义标题栏**与整体 **UI 重构/优化**（控制面板、布局与交互）。
- **界面优化**：深色模式 **黑灰色**主题、主操作按钮在深色下的样式、**设置页**与**圆形图标返回按钮**、首页**倒计时圆环与时间数字放大**等。

### 悬浮窗与菜单

- **新增桌面悬浮窗**；并优化悬浮窗与**托盘/右键菜单**体验。
- **禁用**各 Web 界面内**系统右键菜单**（避免误触）。
- **修复**：从系统关闭应用时的**退出行为**。

### 快捷键与计时

- **空格键**：在**主界面**与**主按钮**一致，用于 **开始 / 暂停 / 继续**（在设置页、编辑剩余时间或焦点在输入框时不触发）。

### 问题修复

- **开始按钮**偶发**无响应**的问题。
