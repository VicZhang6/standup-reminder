# 功能建议 / Feature Ideas

基于对现有代码的分析，以下是按优先级分类的功能建议。

---

## 一、高优先级（实用性强，用户需求明确）

### 1. 久坐统计与健康报告
- 记录每日站立次数、久坐时长、提醒响应速度
- 按日/周/月维度展示统计图表（柱状图、趋势线）
- 目前所有数据仅存 localStorage，增加持久化统计可大幅提升产品价值

### 2. 自定义提醒音效
- 提醒弹窗目前是静默的，容易在专注时错过
- 支持内置几种音效 + 用户自选音频文件
- 可设置音量大小

### 3. 全局快捷键
- 当前 Space 快捷键仅在主窗口获焦时生效
- 通过 Tauri 的 `tauri-plugin-global-shortcut` 注册系统级快捷键
- 用户可自定义快捷键绑定（如 `Ctrl+Shift+S` 开始/暂停）

### 4. 开机自启动
- 使用 `tauri-plugin-autostart` 实现
- 在设置面板增加开关
- 这是桌面工具类应用的基本期望功能

### 5. 多语言支持 (i18n)
- 目前界面全部为中文硬编码
- 至少支持中文/英文切换
- 可用简单的 JSON 翻译文件 + 运行时切换方案

---

## 二、中优先级（提升体验与差异化）

### 6. 休息动作建议
- 提醒弹窗中展示随机的拉伸/活动建议（如「转动脖子」「伸展手臂」「深呼吸 5 次」）
- 配图或简短动画说明
- 可内置一组建议并支持用户自定义

### 7. 智能免打扰模式
- 检测全屏应用（游戏、演示、视频会议）时自动延迟提醒
- 可设置「专注时段」（如 14:00-16:00 不打扰）
- 延迟的提醒在退出免打扰后补发

### 8. 番茄钟模式
- 与现有计时器架构高度契合
- 工作 25 分钟 → 短休息 5 分钟 → 长休息 15 分钟的循环
- 可在「久坐提醒」和「番茄钟」模式间切换
- 统计专注完成次数

### 9. 提醒倒计时样式自定义
- 支持不同的倒计时视觉风格（进度环/数字翻页/沙漏动画等）
- 支持自定义主题色（不限于当前的 indigo）
- 悬浮窗大小可调

### 10. 数据导出
- 将统计数据导出为 CSV/JSON
- 方便用户备份或导入其他健康追踪工具

---

## 三、锦上添花（长期规划）

### 11. 多显示器感知
- 悬浮窗和提醒窗口出现在当前活跃的显示器上
- 记住每个显示器上悬浮窗的位置

### 12. 团队/社交功能
- 同事间共享站立状态（可选的轻量网络功能）
- 团队排行榜或互相提醒
- 适合小型团队健康活动

### 13. 系统通知集成
- 除了全屏弹窗外，增加系统原生通知作为轻量提醒选项
- 使用 `tauri-plugin-notification`
- 用户可选择「全屏提醒」或「系统通知」

### 14. 桌面小组件
- macOS 支持 Widget（需原生开发）
- Windows 支持桌面小组件
- 显示今日统计摘要

### 15. Linux 支持
- 当前仅构建 Windows 和 macOS
- Tauri 天然支持 Linux，增加 CI 构建和测试即可
- 扩大用户群体

---

## 实施建议

| 建议 | 难度 | 涉及模块 |
|------|------|----------|
| 久坐统计 | 中 | 新增统计页面 + localStorage/文件持久化 |
| 提醒音效 | 低 | reminder 窗口 + 设置面板 |
| 全局快捷键 | 低 | Rust 后端 + tauri-plugin-global-shortcut |
| 开机自启 | 低 | Rust 后端 + tauri-plugin-autostart |
| 多语言 | 中 | 全部前端文件 |
| 休息建议 | 低 | reminder 窗口 |
| 免打扰 | 中 | control-renderer + Rust 后端 |
| 番茄钟 | 中 | control-renderer + 新 UI 状态 |
| 系统通知 | 低 | Rust 后端 + tauri-plugin-notification |
| Linux 支持 | 低 | CI/CD 配置 |

建议优先从**开机自启、提醒音效、全局快捷键**入手，这三个功能实现难度低、用户感知强，能快速提升产品完成度。
